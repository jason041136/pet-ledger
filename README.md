# 怪獸小窩｜記帳養成 PWA

自用的寵物養成記帳 App。記一筆帳＝餵一隻怪獸，五隻怪獸的體型隨當月各類支出佔比變化——打開小窩一眼看出錢都流去哪了。

## 五隻怪獸

| 怪獸 | 負責帳目 | 個性 |
|---|---|---|
| 噗嚕 | 手搖飲、零食宵夜、購物、課金儲值 | 惡質史萊姆，餵越多越囂張 |
| 墨墨 | 書籍課程、健身運動 | 書生貓頭鷹 |
| 金金 | 儲蓄、投資 | 守財龍，唯一餵越多越好的 |
| 磚磚 | 房租、水電網路、交通、三餐、日用品 | 沉穩石頭管家 |
| 啾啾 | 聚餐社交、娛樂、旅遊 | 派對鸚鵡，過量會宿醉 |

## 功能

- 3 秒記帳：數字鍵盤 → 點類別 → 完成，怪獸立刻吐槽
- 定期開支：設定「每月幾號扣多少」，到期自動入帳（設定分頁）
- 帳本：按月瀏覽、各怪獸小計、刪除帳目
- 分析：帳本切到「分析」——怪獸佔比圓餅圖、分類排行、近六個月趨勢
- 主題切換：設定 → 外觀，跟隨系統／淺色／深色
- 自訂分類：設定 → 分類管理，新增（自選 emoji、指定餵哪隻怪獸）、改名（✎）、刪除；預設分類也能改
- 匯入／匯出備份：JSON 檔，換手機搬資料用
- 支付方式：每筆帳可標記 現金／悠遊卡／信用卡（可自行新增卡片），帳本與分析都看得到各支付方式的花費
- 悠遊卡餘額：刷卡自動扣、➕加值按鈕、餘額低於 100 提醒、設定頁可校正餘額；加值記錄不算入支出統計
- 台新 Richart 當日方案推薦：小窩會依「今日（或近七天）消費結構」建議切換哪個權益方案（七大方案對照表在 js/data.js 的 RICHART_PLANS／CAT_PLAN，通路異動時自行更新）
- 台詞引擎：台詞依「當月佔比」分三級（被冷落／滿足／失控），會嵌入真實統計數字，台詞庫在 `js/data.js` 的 `DIALOG`，想加梗直接改
- 離線可用（Service Worker）、資料存本機 IndexedDB、可匯出 JSON 備份

## 本機試跑

```
python -m http.server 8642 --directory pet-ledger
```

開 http://localhost:8642

## 裝到手機（免費）

1. 把這個資料夾推上 GitHub，開啟 GitHub Pages（Settings → Pages → main branch）
2. 手機瀏覽器開啟該網址 → Android Chrome 選「加入主畫面」／iPhone Safari 分享 → 「加入主畫面」
3. 之後就像原生 App 一樣全螢幕開啟，離線也能記帳

> 注意：資料存在該裝置的瀏覽器裡。iOS 若太久沒開啟 Safari 網站資料可能被清，記得偶爾在設定分頁匯出備份。

## 改版備忘

- 更新程式後要讓已安裝的手機拿到新版：把 `sw.js` 第一行的 `pet-ledger-v1` 版本號 +1
- 換 AI 生圖：怪獸圖目前是 `js/data.js` 裡的 inline SVG（`PETS[].svg`），換成 `<img>` 指向 PNG 即可，建議每隻準備 平常／開心／肥胖／被冷落 四態
